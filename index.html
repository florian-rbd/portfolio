Qui suis-je ?

Je suis un jeune étudiant de 20 ans en informatique, je suis passionné par le développement web et le développement d'application. Je suis actuellement en formation à l'IUT d'Arles afin d'obtenir mon diplôme du BUT et en alternance au sein de l'entreprise Crossover Technologies. Pour arriver à ce stade, j'ai d'abord obtenu mon BAC général scientifique avec mention, puis j'ai directement commencé mes études supérieures en informatique. J'ai décidé de continuer mes études à l'université d'Arles afin de pouvoir obtenir BUT Informatique. J'ai ensuite décidé de continuer mes études en alternance afin d'acquérir de l' expérience professionnelle et de pouvoir financer mes études. À la fin de mon cursus, je souhaiterais devenir développeur dans une petite entreprise locale où je pourrais m'épanouir et m'améliorer dans ce domaine.

03.

Projets

Pixel Place 2022

Projet personnel HTML CSS JavaScript SQL

Pixel Place est l'un de mes plus gros projet personnel que j'ai réalisé en 2022 suite à un événement communautaire sur le réseau social Reddit. Il s'agit d'un événement collaboratif où chaque utilisateur peut dessiner un pixel sur une grille toute les 5 minutes. J'ai décidé de recréer ce concept au sein du groupe de classe pour pouvoir continuer à en profiter.

Draw with me! 2024

Projet universitaire HTML CSS JavaScript SQL

Draw with me! est un projet réalisé en 2024 dans le cadre d'un projet à réalisé à l'université. Ce projet met à disposition un site depuis lequel il est possible de créer à plusieurs un dessin sur un thème donné. Chaque joueur dispose d'un certain temps pour participer puis passe son tour au prochain pour qu'il continue ce même dessin.

Gestionnaire de mot de passe 2023

Projet universitaire Java

Ce gestionnaire de mot de passe est un projet réalisé en 2022 dans le cadre d'un projet d'une ressource à réalisé à l'université. Cette application bureau laisse la possibilité à l'utilisateur de renseigner ses identifiants et mots de passe afin de ne plus les oublier. Les données enregistrés sont chiffrés dans un fichier local afin de renforcer la sécurité des informations de son utilisateur.

Next'Train 2023

Projet universitaire HTML CSS JavaScript

Next'Train est un projet réalisé en 2022 dans le cadre d'un projet d'une ressource à réalisé à l'université. Le site web permet aux utilisateurs de consulter les prochains trains disponibles selon un départ et une destination donnée. Les données sont récupérés depuis l'API SNCF et affiche sur la carte le trajet et les arrêts du train.

Puissance 4 2023

Projet universitaire HTML CSS JavaScript

Ce puissance 4 a été réalisé en 2023 dans le cadre d'un projet d'une ressource à l'université. Ce jeu permet à deux joueurs de s'affronter sur un puissance 4 classique ou bien seul contre une intelligence artificielle. Le jeu est jouable depuis un site et possède des animations qui rendent le jeu plus agréable à l'oeil.
